## 修改计划

仅需修改 **1 个文件**：

### `src/App.vue`（第 52-59 行区域）

在现有的 H5 原生 tabbar 隐藏规则之后，新增 `uni-page-body` 和 `uni-page-wrapper` 的 overflow 覆盖：

```scss
/* H5 下确保页面内容可滚动 */
/* #ifdef H5 */
uni-page-body,
uni-page-wrapper {
  overflow-y: auto !important;
  -webkit-overflow-scrolling: touch;
}
/* #endif */
```

**注意**：使用条件编译 `/* #ifdef H5 */` 仅在 H5 环境生效，避免影响小程序端。

### 不修改的文件（已符合要求）

- `src/pages/index/index.vue` — `.home` 已有 overflow-y: auto 和 -webkit-overflow-scrolling: touch
- `src/components/GlobalTabbar/GlobalTabbar.vue` — 图标、激活态、中央按钮均已正确
- `src/styles/variables.scss` — 红色调变量已配好
- `src/store/tab.ts` — 状态管理正常
